# Prismatic Raytracer Adventure

A recursive raytracer built on top of a provided OpenGL/MFC graphics framework, developed as a course project.

## Authors

- **Adil Mallick**

## Features

- Recursive raytracing with configurable max depth
- Phong shading: ambient, diffuse, and specular lighting
- Reflections via recursive ray casting
- Hard shadows with shadow ray intersection tests
- Texture mapping (marble, wood plank, world map, sunset)
- Scene includes: mirrored box, brass pyramid, octagonal cylinder, textured floor, and additional non-box shapes
- OpenGL real-time preview alongside the raytraced output

## Build

Open `Project1.sln` in Visual Studio (Windows, MSVC). Build and run — the left panel shows the OpenGL preview and the right panel shows the raytraced result.
